本文介绍一下，python读取图片、保存图片的方法。

百度经验:jingyan.baidu.com

工具/原料

  • 电脑
  • python3.6（anaconda）
  • matplotlib模块
  • opencv模块
  • pillow模块
  • photomosaic模块
百度经验:jingyan.baidu.com

方法/步骤

  1. 1

    用opencv读取图片：

    import cv2

    img = cv2.imread(path)

    print(type(img))

    这里，得到的img是一个图片数组。

    步骤阅读
  2. 2

    用matplotlib打开图片：

    import matplotlib.image as mp

    img = mp.imread(path)

    print(type(img))

    这和cv2读图的效果一样，是数组。

    步骤阅读
  3. 3

    用matplotlib把数组保存为图片：

    mp.imsave('000.png',img)

    用opencv把数组保存为图片：

    cv2.imwrite('000.png',img)

    原理是一样的。

    步骤阅读
  4. 4

    用photomosaic读取图片：

    import photomosaic as pm

    img = pm.imread(path)

    print(type(img))

    也得到数组。

    步骤阅读
  5. 5

    但用photomosaic读取图片得到的数组，与matplotlib读图得到的数组不一样，表现在保存的时候。

    这应该与数组的数据类型有关系。

    步骤阅读
    步骤阅读
  6. 6

    用 pillow读取图片：

    from PIL import Image

    img = Image.open(path)

    print(type(img))

    得到的不是数组。

    步骤阅读
  7. 7

    用pillow保存图片：

    img.save('000.png')

    步骤阅读
    END

matplotlib的另一种保存图片的方法

  • 上面介绍的用matplotlib读图、保存图片的方法，是其中一种。

    可是如果是matplotlib做的图（比如折线图），需要用另一种方法保存：

    import matplotlib.pyplot as plt

    data = [1,3,5,6,9,1,5,8,7,2]

    plt.plot(data,color='g')

    plt.scatter(list(range(10)),data,color='r')

    plt.savefig('000.png')
